VEYRATH PROFESSIONAL WEBSITE V3

This version is a visual rebuild focused on a more premium clothing-brand feel:
- lighter graphite / warm ivory / muted gold / deep oxblood palette
- sharper iOS-style glass navbar, less round UI
- animated fullscreen menu
- refined intro animation
- stronger 3D hero monolith and floating chips
- cleaner cards with beveled corners instead of overly round corners
- SEO metadata, legal pages, admin panel and Supabase-ready files preserved

UPLOAD TO GITHUB PAGES
1. Extract this zip.
2. Upload all files inside the Veyrath-Pro-Website-v3 folder into your GitHub repo root.
3. Replace all old files.
4. Wait 1-2 minutes for GitHub Pages to update.
5. Open index.html, shop.html and admin.html.

SUPABASE SETUP
1. Open supabase-schema.sql in Supabase SQL Editor and run it.
2. Open supabase-config.js.
3. Paste your Supabase URL and anon public key.
4. Set enabled: true.

ADMIN PANEL
Open admin.html.
Default local admin PIN: veyrath-admin

Local mode can add/edit products in your browser and export products.js.
Supabase mode can save products/leads/events to Supabase once configured.

IMPORTANT
The website shows no fake collection names by default. Product sections stay empty until you add products from admin/Supabase or products.js.
