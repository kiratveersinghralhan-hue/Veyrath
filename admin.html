Supabase Setup

  1. Open Supabase → SQL Editor.
  2. Run supabase-schema.sql.
  3. Create an Auth user for yourself.
  4. Add your Auth user ID into admin_users.
  5. Paste Project URL and anon public key in supabase-config.js, then set enabled: true.

Do not paste service_role key inside website files.
